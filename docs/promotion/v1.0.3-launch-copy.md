## 3. V2EX 版

**标题**：开源一个 macOS 状态栏小工具 QuotaMonitor，监控 Kimi / MiniMax / GLM  Coding Plan 配额

用 AI 写代码时最怕 5 小时滚动配额突然耗尽。QuotaMonitor 是一个原生 macOS 状态栏应用，把 Kimi Code、MiniMax、智谱 GLM 三个平台的配额百分比、剩余量、重置倒计时放在菜单栏，看一眼就知道还能写多久。

**主要功能**
- 三平台并行抓取，平均 200ms 内刷新一次
- 智能节流：活跃 30s / 普通 1m / 空闲 5m / 休眠暂停
- 状态栏颜色随最高百分比变化（绿/黄/橙/红）
- 阈值支持 slider 拖动 + 数字框精确输入
- API Key 存 macOS Keychain，显示需 Touch ID / 系统密码
- 系统通知 3 级告警 + Bark 推送到 iPhone

**技术栈**：Swift 5.9 / SwiftUI + AppKit / async-await / actor / Keychain / UNUserNotificationCenter

**开源地址**：https://github.com/cuichaoing/QuotaMonitor
**直接下载**：https://github.com/cuichaoing/QuotaMonitor/releases/tag/v1.0.3

MIT 协议，欢迎 Star / Issue / PR。
