# QuotaMonitor 交付形态报告

> 版本：v1.0.2（2026-06-17）
> 报告类型：B 类（决策辅助，需审阅）
> 适用：用户准备将 QuotaMonitor 交付出去（给朋友/开源/上架）前的形态选择

## 0. TL;DR（一页答案）

**核心问题**：macOS 应用不能像脚本一样 "扔个 .app 就跑"，必须解决"身份信任"问题。

| 交付对象 | 推荐形态 | 工作量 | 体验 |
|---------|---------|--------|------|
| **自己用** | 当前 `build/QuotaMonitor.app`（ad-hoc 签名）| 0h | 直接打开 |
| **朋友/同事** | `.dmg` + ad-hoc 签名 + 安装说明 | 1h | 首次需"右键→打开" |
| **公开下载**（非商店） | `.dmg` + **Developer ID 签名 + Apple 公证** | 5h + ¥688/年 | 任何人无障碍打开 |
| **Mac App Store** | `.pkg` + 沙盒适配 + Apple 审核 | 40h + ¥688/年 | 用户搜得到 |

**给 QuotaMonitor 的推荐**：开源 GitHub → 走 **"公开下载"** 形态（Developer ID 签名 + 公证 + 附 .dmg）。

## 2. macOS 应用 4 种交付形态

### 2.1 形态 A：裸 .app（ad-hoc 签名）
- **结构**：`QuotaMonitor.app` 文件夹
- **签名**：ad-hoc（无身份）
- **分发方式**：直接拷贝 / 邮件附件 / U 盘
- **别人体验**：弹拦截窗口
- **门槛**：0 元 / 0 分钟
- **适用**：自用、给极客朋友、临时分享

### 2.2 形态 B：.dmg / .zip + ad-hoc 签名
- **结构**：`QuotaMonitor-1.0.2.dmg`（磁盘映像，标准 macOS 分发形式）
- **签名**：ad-hoc
- **分发方式**：GitHub Release / 网盘 / 邮件
- **别人体验**：dmg 拖拽到 Applications 体验好，**但首次打开仍被拦截**
- **门槛**：0 元 / 1 小时（学 create-dmg）
- **适用**：开源项目早期、个人项目

### 2.3 形态 C：.dmg + Developer ID 签名 + Apple Notary（推荐）
- **结构**：`QuotaMonitor-1.0.2.dmg` + 内部 `.app` 是 Developer ID 签名 + Apple 公证
- **签名**：Developer ID Application: Your Name (TEAMID)
- **公证**：上传到 Apple notary service，Apple 扫描后盖"已公证"章
- **分发方式**：GitHub Release / 官网 / Homebrew Cask
- **别人体验**：**完全无障碍**（"此 App 已通过 Apple 公证"）
- **门槛**：¥688/年（Apple Developer 个人账号）+ 5h 一次性配置 + 5min/次发布
- **适用**：所有正式公开项目
- **行业标准**：VSCode / Iterm2 / Notion / Obsidian 都用这个

### 2.4 形态 D：Mac App Store
- **结构**：`.pkg`（Package Installer）
- **签名**：Apple 颁发的 Distribution 证书
- **审核**：Apple 人工/自动审核（沙盒、UI、权限、元数据）
- **周期**：7-30 天
- **抽成**：Apple 30%（小开发者可降至 15%）
- **门槛**：¥688/年 + 40h 沙盒适配 + 持续审核沟通
- **适用**：商业产品、需付费分发、需要系统级 API（沙盒限制）

### 2.5 形态对比表

| 形态 | 门槛（钱） | 门槛（时间） | 用户体验 | 适用 |
|------|-----------|------------|---------|------|
| A: 裸 .app | 0 | 0 | 弹拦截 | 自用 |
| B: .dmg + ad-hoc | 0 | 1h | 弹拦截 | 早期开源 |
| **C: .dmg + Dev ID + Notary** | **¥688/年** | **5h + 5min/次** | **无障碍** | **正式开源（推荐）** |
| D: Mac App Store | ¥688/年 | 40h+ | App Store | 商业化 |

## 3. 推荐路径（按用户场景）

### 3.1 用户场景：自用 + 偶尔给朋友
→ **形态 A → 形态 B**

**步骤**：
```bash
# 1. 安装 create-dmg（一次性）
brew install create-dmg

# 2. 确保 .app 已就位
swift build -c release
cp .build/release/QuotaMonitor build/QuotaMonitor.app/Contents/MacOS/QuotaMonitor
codesign --force --deep --sign - build/QuotaMonitor.app

# 3. 修复 Info.plist 版本
/usr/libexec/PlistBuddy -c "Set :CFBundleShortVersionString 1.0.2" build/QuotaMonitor.app/Contents/Info.plist
/usr/libexec/PlistBuddy -c "Set :CFBundleVersion 2" build/QuotaMonitor.app/Contents/Info.plist

# 4. 制作 dmg
create-dmg \
  --volname "QuotaMonitor" \
  --volicon "build/QuotaMonitor.app/Contents/Resources/AppIcon.icns" \
  --window-pos 200 120 \
  --window-size 600 400 \
  --icon-size 100 \
  --icon "QuotaMonitor.app" 175 190 \
  --app-drop-link 425 190 \
  --no-internet-enable \
  "build/QuotaMonitor-1.0.2.dmg" \
  "build/QuotaMonitor.app"
```

**给朋友时附说明**：
> 首次打开如遇拦截，请"右键 → 打开 → 仍要打开"（macOS 安全策略，只这一次）

### 3.2 用户场景：公开下载（GitHub Release）
→ **形态 C（推荐）**

**额外步骤**：

#### (a) 注册 Apple Developer 个人账号
- 网址：https://developer.apple.com/programs/enroll/
- 费用：¥688/年（个人）/ ¥2,888/年（企业）
- 周期：注册当天即可，但**证书签发需要 1-3 天**

#### (b) 在 Xcode / Keychain 制作证书
```bash
# 1. 打开 Xcode → Settings → Accounts → 添加 Apple ID
# 2. 选中账号 → Manage Certificates → + → Developer ID Application
# 3. 证书自动下载到 Keychain
security find-identity -p codesigning -v
# 期望输出：1 valid identity found: "Developer ID Application: Your Name (TEAMID)"
```

#### (c) 修改 .app 签名（用 Developer ID 而非 ad-hoc）
```bash
codesign \
  --force --deep \
  --sign "Developer ID Application: Your Name (TEAMID)" \
  --options runtime \
  --timestamp \
  build/QuotaMonitor.app
```

#### (d) Apple Notary 公证
```bash
# 1. 把 Apple ID 凭证存到 Keychain（一次性）
xcrun notarytool store-credentials "notary-quota" \
  --apple-id "<YOUR_APPLE_ID>" \
  --team-id "TEAMID" \
  --password "app-specific-password"

# 2. 提交 .app 给 Apple 扫描
xcrun notarytool submit build/QuotaMonitor.app \
  --keychain-profile "notary-quota" \
  --wait

# 3. 把公证票据"装订"到 .app
xcrun stapler staple build/QuotaMonitor.app

# 4. 验证
xcrun stapler validate build/QuotaMonitor.app
spctl --assess --type execute --verbose build/QuotaMonitor.app
# 期望输出：build/QuotaMonitor.app: accepted
```

#### (e) 制作 dmg（同样 create-dmg）
同上 3.1 步骤 4

#### (f) 公证 dmg（Apple 建议公证最终 dmg 而非 .app）
```bash
xcrun notarytool submit build/QuotaMonitor-1.0.2.dmg \
  --keychain-profile "notary-quota" \
  --wait
xcrun stapler staple build/QuotaMonitor-1.0.2.dmg
```

#### (g) 发布到 GitHub Release
```bash
gh release create v1.0.2 \
  build/QuotaMonitor-1.0.2.dmg \
  --title "QuotaMonitor v1.0.2" \
  --notes-file CHANGELOG.md
```

### 3.3 用户场景：Mac App Store
→ 形态 D（工作量最大，**不建议**先做）

需额外做：
- 沙盒适配（Keychain 需要 security-scoped bookmarks / 调整 entitlement）
- App Icon 1024x1024 + 多分辨率
- 截图、关键词、类别
- Privacy Manifest（iOS 17+ / macOS 14+ 新规）
- 提交流程 + 应对审核驳回
- 周期：发布后 7-30 天出结果

## 6. 常见陷阱

| 陷阱 | 症状 | 解决 |
|------|------|------|
| **未硬化运行时（hardened runtime）** | `--options runtime` 缺失 | 公证会失败 |
| **未装订（staple）** | 离线电脑打不开 | `xcrun stapler staple` |
| **公证密码** | `notarytool` 用 Apple ID 密码失败 | 需要 https://appleid.apple.com → App-Specific Password |
| **证书 team 不匹配** | "no identity found" | Xcode 登录 Apple ID 后重新签发 |
| **Info.plist 缺字段** | App Store 拒批 | `LSApplicationCategoryType` / `NSHumanReadableCopyright` 等 |
| **dmg 名字带空格** | `create-dmg` 报错 | 引号包裹路径 |
| **未更新 Info.plist 版本号** | 替换 .app 时 macOS 缓存旧版本 | 修改 CFBundleShortVersionString + CFBundleVersion |
| **Intel 用户打不开** | `Mach-O thin (arm64)` | 添加 `--arch x86_64` 构建 universal binary（`lipo`） |
